Font Manager

The Font Manager interface is divided into three main areas*



Please note that while any changes made are applied immediately, any open applications will need to be re-started to reflect those changes.
If Font Manager is open when fonts are installed or removed, it will also need to be re-started.




The Collections area where you can add or remove collections simply by clicking the add or remove buttons, activate entire collections by clicking the checkmark, or de-activate them by clicking the X.

After creating a new collection, to place fonts in it just drag and drop them from the fonts column onto your newly created collection.


The Fonts area where you can select which font to preview, activate or de-activate fonts and also search through the list of fonts.




This is the Preview area where you can select which style of font to display by using the combobox at the top, the size of the font used in the preview can be changed with the slider at the bottom, or for more control you can use the spinbutton at the top right of the preview area.
You can also choose to enter your own text to see how it would look in the selected font.

Here you can also get information about the currently selected font, such as file location and size.
If gnome-font-viewer is installed on your system it will be used to show any available descriptions, copyrights, etc. for the selected font.



* Font Manager has been designed to be as simple and intuitive as possible, hopefully you find this to be true, but if not please let us know how it may be improved.
