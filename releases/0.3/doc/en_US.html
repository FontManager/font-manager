Font Manager

 A font management application for the GNOME desktop


Please note that while changes made using Font Manager are applied immediately,
any running applications will need to be re-started to reflect those changes.

If Font Manager is open when fonts are installed or removed, it will also need to be re-started.





The Collections area where fonts can be grouped into collections and activated or de-activated all at once.

After creating a new collection, to place fonts in it just drag and drop them from the fonts column onto the newly created collection.





The Fonts area allows the user to select which font to preview, activate or de-activate fonts and also search through the list of fonts.

Selecting multiple fonts can be done by clicking and dragging, or by using shift / control key combinations.






This is the Preview area where font styles can be previewed individually or compared in a list, selecting a font style is done by using the combobox at the top, the size of the font used in the preview can be changed with the slider at the bottom, or for more control the spinbutton at the top right of the preview area.

Custom text allows the user to see how entered text would look in the selected font, compare mode also allows changing font / background colors.


Optional features :

If gnome-font-viewer is installed on the system, the information icon at the top left of the preview area will display any available descriptions, copyrights, etc. for the selected font.

If file-roller is available, exporting collections to an archive becomes an option.


*Font Manager has been designed to be as simple and intuitive as possible, hopefully you find this to be true, if not please let us know how it may be improved.
